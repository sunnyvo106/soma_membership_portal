# Soma Membership Portal  
## Epic & Feature Backlog

This document outlines the **high-level epics and features** for the Soma Membership Portal.  
It bridges business objectives (BRD) and delivery-level user stories (Agile backlog).

---

## Epic E1: Member Self-Registration
Enable users to register, verify identity, and gain secure access to the membership portal.

### Features
- **F1.1 Email-Based Registration**  
  Allow users to register using email and password.

- **F1.2 Email Verification**  
  Verify user email via time-bound confirmation link.

- **F1.3 Identity Verification**  
  Enable upload and validation of ID and visa documents.

- **F1.4 Profile Creation**  
  Capture and store personal and contact details.

---

## Epic E2: Website & Public Information
Provide public-facing pages for visitors and prospective members.

### Features
- **F2.1 Landing Page**  
  Display headline message, introduction video, and testimonials.

- **F2.2 About Us Page**  
  Provide organisational background, achievements, and values.

- **F2.3 Contact Us Page**  
  Allow users to submit enquiries via a contact form.

---

## Epic E3: Membership Portal Management
Allow members to manage profiles, payments, and donations.

### Features
- **F3.1 Secure Login & Authentication**  
  Enable secure access to the membership portal.

- **F3.2 Profile Management**  
  Allow members to view and manage personal information.

- **F3.3 E-Wallet Setup**  
  Enable members to create and link an e-wallet.

- **F3.4 Donations & Payments**  
  Allow members to make donations using linked payment methods.

---

## Epic E4: Community Projects
Enable visibility, management, and governance of community initiatives.

### Features
- **F4.1 Manager Role Request**  
  Allow members to request elevated access.

- **F4.2 Project Content Upload**  
  Allow managers to upload project documents and images.

- **F4.3 Approval Workflow**  
  Enable administrators to review and approve project content.

- **F4.4 Public Project Visibility**  
  Display approved projects to members and visitors.

---

## Epic E5: Loan Management
Support financial assistance through a structured loan request process.

### Features
- **F5.1 Loan Request Submission**  
  Allow members to submit loan applications.

- **F5.2 Loan Review & Notification**  
  Enable administrators to review requests and notify members.
